portable system

在建置可攜系統前須在資料夾中(例如:CD2020)先建立一data的資料夾供之後放置相關必要檔案和要分割的磁碟內容

所需的檔案:

Portable Git:以使用相關的git指令來操作

Python:使能執行相關python檔案和在SciTE中進行Python程式的編寫

SciTE:能利用此文字編輯器來進行不同程式語言的撰寫(e.g. Python、html)

MSYS2:能在windows環境下建立類linux環境

get-pip.py:使用Python時會需要相關套件，用pip來做管理較為方便

建立可攜系統步驟:

將下載後的Portable Git檔案放置在CD2020\data\portablegit資料夾中

將下載後的Python檔案放置在CD2020\data\py382資料夾中

將下載後的SciTE檔案放置在CD2020\data\wscite432資料夾中

將下載後的MSYS2檔案放置在CD2020\data\msys64資料夾中

將下載後的get-pip.py檔案放置在CD2020\data資料夾中並執行

在CD2020資料夾中建立一start.bat檔案，內容為:

@echo off
set Disk=y
subst %Disk%: "data"

%Disk%:

set HomePath=%Disk%:\home_mdecourse
set HomeDrive=%Disk%:\home_mdecourse
set Home=%Disk%:\home_mdecourse
set USERPROFILE=%Disk%:\home_mdecourse

REM 將系統 Python 程式的 io 設為 utf-8
set PYTHONIOENCODING="utf-8"

REM for putty
Set GIT_HOME=%CDisk%:\portablegit\bin\
Set GIT_SSH=%Disk%:\putty\plink.exe

set PYTHONPATH=%Disk%:\py382\DLLs;%Disk%:\py382\Lib;%Disk%:\py382\Lib\site-packages;
set PYTHONHOME=%Disk%:\py382

set path_python=%Disk%:\py382;%Disk%:\py382\Scripts;
set path_msys2=%Disk%:\msys64\mingw64\bin;
set path_git=%Disk%:\portablegit\bin;
set path_tcc=%Disk%:\tcc;

path=%Disk%:;%path_python%;%path_msys2%;%path_git%;%path_tcc%;

start /MIN cmd.exe
start /MIN cmd.exe
start /MIN cmd.exe
start /MIN cmd.exe

start /MIN %Disk%:\wscite432\wscite\SciTE.exe
start /MIN %Disk%:\wscite432\wscite\SciTE.exe

Exit

主要為設置路徑至上述檔案放置的資料夾、建立一分割磁碟Y、開啟命令列和開啟SciTE

再建立一stop.bat檔案供關閉用start.bat檔案開啟的相關內容

@echo off
set Disk=y
path=%PATH%;

taskkill /IM python.exe /F
taskkill /IM pythonw.exe /F
taskkill /IM scite.exe /F
taskkill /IM ShareX.exe /F

REM 終止虛擬硬碟與目錄的對應
subst %Disk%: /D
REM 關閉 cmd 指令視窗
taskkill /IM cmd.exe /F

EXIT
